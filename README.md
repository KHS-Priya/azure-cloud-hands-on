This project contains hands-on practice with Azure using Azure CLI, Azure Portal, and basic services.

## What This Includes

- ✅ Virtual Machine (VM) creation with CLI and Portal
- ✅ Resource Group and VNet setup
- ✅ Azure Storage Account basics

- ✅ Azure CLI Commands used

To showcase hands-on knowledge of Azure infrastructure setup, resource management, and access control — aligned with AZ-104 certification objectives.

## Skills Demonstrated

- Azure Resource Management
- Azure CLI and Portal usage
- Networking (VNet, NSG)
- Identity and Access (IAM, RBAC)
- Infrastructure documentation
